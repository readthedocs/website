Take the confusion out of docs as code

Read the Docs is the go-to solution for open source projects and businesses for docs as code hosting. Use our standard process to build, publish, and host your documentation. Giving you all the features you care about without having to reinvent the wheel.

Write documentation using a rich ecosystem of tools (MkDocs, Docusaurus, etc).
Version your documents with Git.
Test your documentation builds correctly.
Preview the changes before publishing.
Automate the publication process.

Read the Docs is the "all in one" solution for docs as code.
